# Projeto Final ATEC - Aplicação de Agendamento

Este é um projeto full-stack que consiste numa API RESTful desenvolvida com Python/FastAPI e um cliente web desenvolvido com React/TypeScript.

## Tecnologias Utilizadas

### Backend

- **Python 3.12+**: Linguagem de programação principal.
- **FastAPI**: Framework web de alta performance para a construção de APIs.
- **SQLAlchemy**: ORM (Object Relational Mapper) para interação com a base de dados.
- **Pydantic**: Para validação de dados e gestão de configurações.
- **Uvicorn**: Servidor ASGI para executar a aplicação FastAPI.
- **SQLite**: Base de dados utilizada no ambiente de desenvolvimento.

### Frontend

- **React**: Biblioteca JavaScript para a construção de interfaces de utilizador.
- **TypeScript**: Superset de JavaScript que adiciona tipagem estática.
- **Vite**: Ferramenta de build moderna para um desenvolvimento frontend mais rápido.
- **Axios**: Cliente HTTP baseado em Promises para realizar requisições à API.
- **Bootstrap**: Framework de CSS para a estilização de componentes.

## Como Executar

O projeto está dividido em duas pastas principais: `backend` e `frontend`. Cada uma contém as suas próprias instruções detalhadas de instalação e execução.

Para começar, por favor, siga os guias em cada diretório:

- **Para o Backend (API):** backend/readme.md
- **Para o Frontend (Aplicação Cliente):** frontend/README.md
